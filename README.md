
# Trilogy Blog Page

## Description

The Trilogy Blog Page is a simple and clean web interface designed to showcase a collection of blog posts in a structured card layout. It allows users to easily browse through different blogs and access their content with smooth navigation, focusing on clarity and user-friendly design.
- Three sections the user can see. Featuring the trio members:
  -> Dishti 
  -> Mallika
  -> Palak
